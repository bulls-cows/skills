# 技能基础模板

本文档是创建新技能的基础模板，适用于大多数通用技能场景。

```markdown
---
name: skill-name
description: >
  精确描述技能做什么以及何时使用。必须是单一、清晰的任务边界。
  包含触发条件说明。
---

## When to use
- 用户要求 X 时
- 涉及 Y 上下文时
- 触发条件 3

## Don't use when
- 用户要求直接实现新功能时
- 简单的 Z 操作时
- 不应触发的场景 3

## Steps
1. 第一步：描述做什么，为什么
2. 第二步：描述下一步操作
3. 第三步：继续描述后续步骤
4. 最后一步：输出结果

## Output contract
- 输出项 1：描述
- 输出项 2：描述
- 输出项 3：描述
```

## 使用说明

1. 将 `skill-name` 替换为实际的技能名称
2. 使用小写、短横线分隔的命名方式
3. 优先使用动宾结构，如 `create-rule`、`read-pdf`
4. 名词使用单数形式，如 `create-skill` 而非 `create-skills`
