# PDF 读取类技能模板

本文档是创建 PDF 读取类技能的模板，适用于从 PDF 文档中提取内容的场景。

```markdown
---
name: read-pdf
description: >
  读取 PDF 文件内容。用于：用户想要提取 PDF 文档中的文本、图片、表格等内容。
---

## When to use
- 用户提到"读取 PDF"、"解析 PDF"
- 用户需要从 PDF 文件中提取信息
- 用户提到 PDF 文件路径并要求分析
- 用户需要处理 PDF 格式的文档

## Don't use when
- 用户读取的是非 PDF 文件
- 用户只是想查看 PDF 文件列表
- 用户要求编辑 PDF 文件
- 用户要求创建新的 PDF

## Steps
1. 确认 PDF 文件路径存在且可访问
2. 使用 PDF 解析工具提取内容：
   - 文本内容
   - 图片（如有）
   - 表格数据（如有）
3. 结构化输出提取的信息
4. 如有需要，进行内容摘要或分析

## Output contract

### 文件信息
- 文件路径
- 页数
- 文件大小

### 文本内容
- 按页面组织的文本内容
- 关键段落摘要

### 图片列表（如有）
- 图片位置（页码）
- 图片描述或 OCR 文本

### 表格数据（如有）
- 表格标题
- 表格数据（结构化格式）
```

## 使用说明

1. 此模板适用于需要读取 PDF 的技能
2. 可根据实际需求调整输出格式
3. 如需 OCR 功能，需在 Steps 中明确说明
