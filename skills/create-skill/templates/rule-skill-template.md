# 规则类技能模板

本文档是创建规则（Rule）类技能的模板，适用于定义 Agent 行为约束、输出格式要求等场景。

```markdown
---
name: create-rule
description: >
  创建或更新 Agent 规则文件。用于：用户想要为特定代理定义行为规则、
  约束条件、输出格式要求等。
---

## When to use
- 用户提到"创建规则"、"添加规则"
- 用户想要定义 Agent 的行为约束
- 用户想要规范 Agent 的输出格式
- 用户提到"规则文件"、"行为约束"

## Don't use when
- 用户只是想创建普通文档
- 用户想要创建 Skill 文件
- 用户要求实现具体功能

## Steps
1. 确定规则名称和目标 Agent
2. 询问用户需要定义的约束类型：
   - 行为约束（允许/禁止的操作）
   - 输出格式要求
   - 触发条件
3. 定义规则内容和约束
4. 创建或更新 `.agents/rules/` 目录下的规则文件
5. 验证规则格式正确性

## Output contract
- 规则文件路径
- 规则内容摘要
- 生效范围说明
```

## 规则文件示例

```markdown
# Agent 行为规则

## 允许的操作
- 读取项目文件
- 修改指定目录下的文件
- 执行构建和测试命令

## 禁止的操作
- 修改配置文件
- 提交代码到版本控制
- 访问外部网络

## 输出格式
- 所有响应使用中文
- 代码变更需要说明原因
- 错误信息需要包含解决方案
```
