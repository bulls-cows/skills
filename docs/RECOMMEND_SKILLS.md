### Superpowers

| ------------------------------ | -------------------------------------------------------------- | ------------------------------------------------------------------------ |
| brainstorming                  | 任何创造性工作之前使用——创建功能、构建组件、添加功能或修改行为 | `npx skills add obra/superpowers --skill brainstorming`                  |
| writing-plans                  | 多步骤任务规范或需求，在接触代码之前使用                       | `npx skills add obra/superpowers --skill writing-plans`                  |
